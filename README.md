# 🫀 Heart Disease Prediction Web App
This project is a web-based application that predicts the risk of heart disease using a trained Machine Learning model. It is built with Python, Flask, and HTML, and provides a simple interface for users to input their health information and receive instant predictions along with a downloadable report.

## 🔧 Technologies Used
- Python
- Flask
- HTML/CSS
- Scikit-learn (Random Forest Classifier)
- SQLite (for user data)
- Joblib (for model storage)

## 📁 Project Structure
- `app.py` – Main backend application
- `templates/` – HTML files for frontend UI
- `HeartDisease.csv` – Dataset used for model training
- `.joblib` – Trained ML model files
- `userdb.db` – SQLite database for authentication
- `.pdf` – Generated reports
- `.ipynb` – Jupyter notebook for model development

## 🚀 How to Run
1. Clone this repository
2. Install dependencies

## ✅ Features
- User login/signup authentication
- Health data input form
- Heart disease risk prediction using ML
- PDF report generation
- Easy-to-use web interface
